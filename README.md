# birds

Personal birding app (needs/targets + travel planner + gallery) at birds.gaylon.photos.

**Authoritative spec**: `docs/birds-app-design.md`

Follows patterns from madonnahist (SvelteKit, Postgres, droplet co-location, auth, deploy, test isolation, no-Tailwind CSS, etc.). Copy/adapt from the madonnahist source tree at the paths noted in the design doc.

Quick start (after setup): see the design doc "Commands" and "Local Test Safety" sections.
